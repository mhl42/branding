# MHL · 42 — Brand Kit

Visual identity for **Mostly Harmless** ([mhl42.ai](https://mhl42.ai)). Element-42 logo system, Inter typography, 3-color palette.

Open `brand-sheet.html` in a browser for an interactive index of every asset with click-to-download.

## Typography

[Inter](https://rsms.me/inter/) by Rasmus Andersson — open source (SIL Open Font License 1.1). Every text element in the SVGs is **outlined to vector paths**, so files render identically anywhere with no font dependency.

## Usage rules

**Do** use the mark with the phrase on first contact (web hero, deck cover, business card front), then drop the phrase for repeat appearances. Give the tile clear space equal to the height of the "42" caption. Use the gold accent for one element per surface.

**Don't** recolor the mark with anything but ink, paper, or gold. Don't add shadows. Don't put the gold dot anywhere except the top-right of the tile. Don't stretch, rotate, or fill the tile interior — it's always an outline.
